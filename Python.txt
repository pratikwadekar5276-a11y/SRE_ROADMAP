- DAY 18: File Handling Basics (Part 1): How to open, read, and close text files securely.
File Handling in Python means reading data from files, writing data to files, and modifying file contents stored on disk.
Before we can perform any operation on file we must first need to open it.
Python provides the open() function to open file.
It takes two argumets first name of the file and second is mode of file the file in which you want to perfom any operation
Closing file is very important in the scenario of append and write becuase it will not write data in that file if you dont close the file
modes in file handeling
r - read - opens the file and gives error if file does not exist

test = f.read()
print(test)
f.close()
f.read()        # Entire file
f.readline()   # Single line
f.readlines()  # All lines as list

w - write - writes - w mode overwrites the existing content and creates a file if not exist
f = open("myfile.txt", "w")
f.write("content")
f.close()

a - append - add content at the end of file
f = open("myfile.txt", "a")
f.write("content")
f.close()

if you use with keyword then you dont need to close the file
with open("myfile.txt", "a/w") as f
