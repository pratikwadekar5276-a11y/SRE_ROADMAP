Inode in linux is a data strucuture that stores metadata about file or Directory

/ - root directory

In Linux a file descriptor is an integer that represents an open file. There are three standerd file descriptors
1 . standerd input file descriptor 0
2 . standerd output file descriptor 1
3 . standerd error file descriptor 2
